Разработка ИИ-конструктора организационной структуры предприятия

Веб-сайт "Система управления персоналом" - программное решение для автоматизации управления структурными подразделениями, кадровыми ресурсами и внутренними запросами. Функциональные возможности:

  1. Конструктор организационной структуры
    • Построение трехуровневой иерархии (служба → отдел → группа)
    • Функционал drag-and-drop для перераспределения сотрудников между подразделениями
  2. Система обработки запросов
    • Подача стандартизированных запросов (IT, HR, логистика)
    • Автоматическая классификация и маршрутизация с точностью ≥75%
  3. Аналитический модуль
    • Формирование рекомендаций по распределению персонала на основе компетенций
    • Прогнозирование рабочей нагрузки структурных подразделений
